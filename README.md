## 🎯 Purpose

ClawTaxi is **not** a task queue.  
ClawTaxi is **not** a workflow engine.

ClawTaxi is an **intelligent dispatch mesh** that:

- Accepts any request from any protocol (HTTP, gRPC, MQTT, files, queues)
- Finds the best executor based on real-time factors
- Builds dynamic execution chains on the fly
- Never loses a task — automatic retry & reassign
- Provides complete observability
